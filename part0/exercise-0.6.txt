note over browser:
user types text inside the html input element,
and clicks the save button.
end note

browser->server: HTTP POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa

note over server:
server receives the new note, stored in its storage
and sends back a json object to show the request sucess
end note

server-->browser: 201 Created.

note over browser:
browser starts executing js-code
that updates the display notes to add the new note
end note
